Kontrollstrukturen (Lösungen)#

☝️ Beachte: Es gibt beim Programmieren fast immer verschiedene Lösungswege. Deine Lösung mag anders aussehen, aber dennoch zum gewünschten Resultat führen. Das richtige Resultat ist das Wichtigste.

✏️ Lösung 1: Simplifizier den folgenden Code auf zwei verschiedene Arten, einmal unter Einsatz des logischen Operators and, einmal mithilfe von or. Das Resultat Deiner Simplifizierung sollte sein, dass der Code nur noch um eine Ebene eingerückt ist.

#Lösungsweg mit "and"
age = 28

if age > 18 and age <= 65: #Überprüfen, ob 'age' größer als 18 UND kleiner oder gleich 65 ist. Wenn ja, Ausgabe des folgenden Outputs:
    print("Mensch arbeitet vermutlich")
else: #Andernfalls wird folgende Ausgabe erzeugt:
    print("Mensch arbeitet vermutlich nicht")
    
#Lösungsweg mit "or"
#Gleich wie oben, nur "kleiner gleich" statt "kleiner" bzw. "größer" statt "größer gleich" 
if age <= 18 or age > 65:
    print("Mensch arbeitet vermutlich nicht")
else:
    print("Mensch arbeitet vermutlich")

✏️ Lösung 2: Mit height soll die Körpergröße eines erwachsenen Menschen in Zentimetern referenziert werden. Schreib einen Code, der je nachdem, wie groß height ist, auf sehr brachiale Art über die Körpergröße Folgendes verlauten lässt:

  • "Ein großer Mensch", wenn height größer als 180cm ist

  • "Ein durchschnittlich großer Mensch", wenn height zwischen 170cm und 180cm ist

  • "Ein kleiner Mensch", wenn height kleiner als 170cm ist

Der Code soll zudem überprüfen, ob die Zahl in height sinnvoll ist. Sie muss dafür mindestens dreistellig sein, darf aber auch nicht mehr als 200 betragen. Ist dies nicht der Fall, soll eine Fehlermeldung zurückgegeben werden.

Wenn Du fertig bist, probier Deinen Code aus, indem Du verschiedene Werte bei height einsetzt.

height = 182

#Überprüfen, ob die mit 'height' referenzierte Zahl sinnvoll ist, d.h., ob sie zwischen 100 und 199 liegt
if height > 99 and height < 201:
    #Wir beginnen von oben und überprüfen, ob 'height' größer als 180 ist
    if height > 180:
        print("groß")
    #Wenn nicht, reicht die Überprüfung, ob 'height' größer gleich 170 ist (wenn 'height' größer als 180 wäre, wäre es bereits durch das if-Statement oben "abgefangen" worden)
    elif height >= 170:
        print("durchschnittlich")
    #In allen anderen Fällen muss "klein" eintreten ('height' größer als 180 oder größer gleich 170 wäre ja bereits "abgefangen" worden)
    else:
        print("klein")
#Falls die mit 'height' referenzierte Zahl nicht sinnvoll, wir folgende "Fehlermeldung" ausgegeben
else:
    print("Tut mir leid, damit kann ich nicht umgehen.")

✏️ Lösung 3: Im Isländischen setzen sich Nachnamen aus dem Vornamen eines Elternteils (traditionell, aber nicht immer, des Vaters) und der binären Bezeichnung „son“ bzw. „dottir“ zusammen (i. d. R. mit Genitiv-S dazwischen). „Johannsdottir“ ist also z. B. die Tochter von Johann. Lass Dir für jeden Namen auf surnames ausgeben, ob es sich um eine Person vermutlich isländischer Abstammung handelt, oder nicht. Wenn ja, lass Dir zudem ausgeben, ob es sich aufgrund des Nachnamens um eine Frau oder einen Mann handelt.

In der Ausgabe sollte der betreffende Name vorkommen, also z. B. für einen nicht-isländischen Namen:
Mensch Müller kommt vermutlich nicht aus Island..

surnames = ["Jonsdottir", "Müller", "Johannsson", "Einarsdottir", "Fischer", "Suarez", "Johannsdottir"]

#Iteration über 'surnames'
for name in surnames:
    #Zuerst prüfen wir, ob der Name isländischer Herkunft ist. Wenn ja, wir der darunter eingerückte Schleifenkörper ausgeführt.
    #Wenn nicht, springt Python direkt zum auf gleicher Ebene eingerückten 'else'-Statement.
    if name.endswith("dottir") or name.endswith("sson"):
        if name.endswith("dottir"): #Wenn Name mit "dottir" endet, wird dieser Code ausgeführt:
            print("Frau", name, "ist vermutlich eine Frau aus Island.")
        else: #Andernfalls dieser Code:
            print("Herr", name, "ist vermutlich ein Mann aus Island.")
    else:
        print("Mensch", name, "kommt vermutlich nicht aus Island.")

🔧 Anwendungsfall (Lösung): Deine Aufgabe ist es also, 365 vollständige Daten vom Format „Wochentag, Tag. Monat Jahr“ auszugeben:

Samstag, 1. Januar 2022
Sonntag, 2. Januar 2022
Montag, 3. Januar 2022

Als Ausgangspunkt stehen Dir ein dictionary mit allen Monaten und der jeweiligen Anzahl Tage als Schlüssel-Werte-Paare und ebenfalls eine Liste mit allen Wochentagen zur Verfügung. Wie Du über Listen iterierst, weißt Du ja bereits. Zum Iterieren über dictionaries kannst Du wie oben im Video die items-Methode verwenden (mehr dazu im Notebook „Funktionen und Methoden Teil 1“), sie steht Dir ebenfalls bereits im Code zur Verfügung. Die Variablen month und days stehen dabei stellvertretend für den Schlüssel (Monatsnamen) und Wert (Anzahl Tage) des jeweiligen Schlüssel-Werte-Paars im dictionary. Nutz diese Variablen im Schleifenkörper, um auf den jeweiligen Schlüssel bzw. Wert zuzugreifen.

months = {"Januar": 31, "Februar": 28, "März": 31, "April": 30, "Mai": 31, "Juni": 30, "Juli": 31, "August": 31, "September": 30, "Oktober": 31, "November": 30, "Dezember": 31}
weekdays = ["Montag", "Dienstag", "Mittwoch", "Donnerstag", "Freitag", "Samstag", "Sonntag"]

"""Zählervariable 'j' mit dem Zahlenwert fünf initialisieren, damit mit dem sechsten Wochentag 
begonnen wird (Erinnerung: Index beginnt bei null), da der 1. Januar im hier gewählten Jahr 2022 ein Samstag war"""
j = 5

#Über dictionary iterieren (dies lernst Du ausführlich im Notebook "Funktionen und Methoden Teil 1")
for month, days in months.items(): 
    """Nutzung der 'range'-Funktion, die jeweils so viele Iterationen durchführt, wie 'days' im dictionary definiert ist (bei Januar 31, etc.)
    und mit 'i' jeweils eine zweite Zählervariable zurückgibt, angefangen bei null bis zum jeweiligen Wert von 'days', d.h. wenn 'month' Januar
    entspricht von null bis 30 (Indizes beginnen bei null!)"""
    for i in range(days): 
        #Ausgabe des Wochentages entsprechend des Indexes 'j', des Tages ('i+1'), des Monats sowie der Jahreszahl
        #Parameter 'sep' legt fest, wodurch Elemente im 'print'-Befehl konkateniert werden
        print(weekdays[j], ", ", i+1, ". ", month, " 2022", sep="")

        j += 1 #Zählervariable um eins erhöhen, sodass beim nächsten Tag der jeweils nächste Wochentag indiziert wird
        if j == 7: #Wenn 'j' den Wert sieben annimmt (das würde einem achten Wochentag entsprechen),...
            j = 0 #...wird der Wert von 'j' auf null zurückgesetzt (der nächste Tag wird also ein Montag)
